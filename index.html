JavaScript

Maak gebruik van JavaScript om animaties te maken.

We kunnen met JavaScript CSS classes toevoegen of verwijderen om animaties te starten.

We kunnen echter ook veel verder gaan, door zelf animaties te maken met JavaScript. Dit kan door bijvoorbeeld de requestAnimationFrame functie te gebruiken. In combinatie met een canvas element kan je hiermee complexe animaties maken.

We kunnen het canvas element ook inzetten om moderne 3d games en animaties te maken. Dit kan door gebruik te maken van de WebGL API. We kunnen hiermee 3d objecten renderen en deze animeren. Deze techniek wordt veel gebruikt voor het maken van games en interactieve 3d visualisaties. Voorbeelden van dit soort visualisaties kun je vinden op threejs.org.
